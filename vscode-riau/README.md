# Riau VS Code Extension

Official Visual Studio Code extension for the Riau Programming Language.

## Features

- **Syntax Highlighting** - Full syntax highlighting for `.riau` files
- **File Association** - Automatic recognition of `.riau` files
- **Bracket Matching** - Auto-closing brackets and quotes
- **Comment Support** - Line comments with `//`
- **Code Folding** - Fold code blocks for better navigation

## Installation

### From Source

1. Clone this repository
2. Open in VS Code
3. Press `F5` to launch Extension Development Host
4. Open a `.riau` file to test

### From Marketplace (Coming Soon)

```bash
code --install-extension riau
```

## Supported Features

### Syntax Highlighting

- Keywords: `let`, `fn`, `if`, `else`, `for`, `in`, `while`, `return`, `try`, `catch`
- Built-in Functions: `print`, `log`, `env`, `type_of`, `len`, etc.
- Operators: `+`, `-`, `*`, `/`, `==`, `!=`, `&&`, `||`, etc.
- Literals: strings, numbers, booleans, null

### Auto-Completion (Coming Soon)

- Built-in functions
- Keywords
- Variables in scope

### Debugging (Coming Soon)

- Breakpoints
- Step through code
- Variable inspection

## Example

```riau
// Hello World
print("Hello from Riau!")

// Web Application
print("Content-Type: text/html\n\n")
print("<h1>Dynamic Web with Riau!</h1>")
```

## Requirements

- VS Code 1.60.0 or higher
- Riau compiler installed (optional, for running code)
