# portfolio

Riau web application created with `init-web.bat`

## Quick Start

1. Deploy to Apache/Nginx
2. Visit `http://localhost/portfolio/`

## Structure

- `index.riau` - Homepage
- `about.riau` - About page
- `api/example.riau` - API endpoint
- `public/` - Static assets
